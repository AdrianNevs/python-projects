# Product Manager (CLI)

Projeto em Python para gerenciamento de produtos no terminal.

# Funcionalidades
- Adicionar produtos
- validação do nome
- Listar produtos
- Remover produtos por ID
- Buscar produtos usando Generator
- Cálculo de preço com taxa

# Tecnologias e conceitos usados
- Python
- raplace
- Organização em múltiplos arquivos
- List comprehension
- Generator
- Closure para cálculo de taxa
- Tratamento de erros (try/except)

# Como executar
1. Abra o terminal na pasta do projeto
2. Execute:
   python main.py


# estrutura do produto
{
     "id": int,
     "name": str,
     "price": float,
     "tax": float
}

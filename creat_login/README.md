# Sistema de Cadastro de Usuários (Python)

Um sistema simples de cadastro de usuários desenvolvido em Python para praticar lógica de programação, estruturas de dados e validações.

# Descrição

Este projeto simula um sistema básico de criação de usuários, onde é possível cadastrar pessoas informando nome e idade.
O sistema impede nomes duplicados e trata entradas inválidas para idade.

## ⚙️ Funcionalidades

* Cadastro de novos usuários
* Validação de nomes duplicados
* Validação de idade (apenas números inteiros)
* Tratamento de erros com `try/except`

Praticar conceitos fundamentais de programação, incluindo:

* Funções
* Listas e dicionários
* Estruturas condicionais
* List comprehension
* Validação de dados

Desenvolvido como parte dos estudos em Análise e Desenvolvimento de Sistemas.
